# Football Training App - ფეხბურთის ვარჯიშების აპლიკაცია

ეს არის სასწავლო პროექტი, რომელიც შექმნილია ფინალური გამოცდისთვის. აპლიკაცია ეხმარება ფეხბურთელებს ვარჯიშების დაგეგმვასა და შედეგების აღრიცხვაში.

## ტექნიკური დეტალები

*   **ენა:** Kotlin
*   **UI ჩარჩო:** Jetpack Compose (Modern Toolkit, არ არის გამოყენებული `findViewById`)
*   **არქიტექტურა:** MVVM (Model-View-ViewModel)
*   **მონაცემთა ბაზა:** Room Database (ლოკალური მონაცემების შესანახად)
*   **ნავიგაცია:** Navigation Compose (Drawer Menu-თი)
*   **ახალი ფუნქციონალი:** Photo Picker (ActivityResultContracts.GetContent) - მედია ფაილების ასარჩევად.
*   **Dependency Management:** Version Catalog (libs.versions.toml)

## ფუნქციონალი

1.  **ვარჯიშების სია (Home):** აჩვენებს ხელმისაწვდომ ვარჯიშებს (Dribbling, Shooting, Physical).
2.  **დეტალური ინსტრუქცია:** თითოეულ ვარჯიშს აქვს თავისი აღწერა.
3.  **შედეგების ატვირთვა:** მომხმარებელს შეუძლია შეიყვანოს თავისი შედეგი და მიაბას ფოტო ან ვიდეო მასალა (Photo Picker-ის გამოყენებით).
4.  **შედეგების ბორდი:** ყველა შენახული შედეგი აისახება ცალკე ეკრანზე.
5.  **პროფილი:** მომხმარებლის ზოგადი ინფორმაცია და სტატისტიკა.
6.  **მენიუ:** ნავიგაცია ეკრანებს შორის ხორციელდება Side Drawer მენიუს მეშვეობით.

## პროექტის სტრუქტურა

*   `data/`: შეიცავს Room ბაზას, DAO-ებს და Repository-ს.
*   `model/`: მონაცემთა კლასები (Entities).
*   `ui/viewmodel/`: ბიზნეს ლოგიკა.
*   `ui/screens/`: Compose ეკრანები.
*   `ui/navigation/`: ნავიგაციის კონფიგურაცია.
